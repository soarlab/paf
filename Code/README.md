# CAV 2020 - PAF Readme File

* Install System requirements: 'sudo apt-get install ???' for each package in `systemRequirements.txt`
* Install the `Python3` requirements with the following:
```
cat Requirements.txt | xargs -n 1 pip3 install
```
* Customize `src/main.py` then run: `python3.5 src/main.py`
